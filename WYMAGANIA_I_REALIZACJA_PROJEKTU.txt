MAJSTERPRO MARKET 2026
Wymagania projektu i zrealizowane elementy

Temat projektu:
Aplikacja mobilna sklepu budowlano-remontowego MajsterPro Market.
Projekt zawiera aplikacje mobilna React Native oraz backend ASP.NET Core Web API z baza SQL Server.


1. CRUD DLA 10 KLAS / TABEL

Wymaganie nauczyciela:
Obsluzony w pelni CRUD (Create, Read, Update, Delete) dla 10 klas/tabel.

Status:
SPELNIONE.

Zrealizowane tabele / klasy:
1. UnitOfMeasurement - jednostki miary
2. Category - kategorie produktow
3. Client - klienci
4. Worker - pracownicy / administratorzy
5. Supplier - dostawcy
6. Brand - marki
7. Warehouse - magazyny
8. Item - produkty
9. Order - zamowienia
10. OrderItem - pozycje zamowien

CRUD jest dostepny w API przez endpointy:
- /api/UnitOfMeasurement
- /api/Category
- /api/Client
- /api/Worker
- /api/Supplier
- /api/Brand
- /api/Warehouse
- /api/Item
- /api/Order
- /api/OrderItem

CRUD jest tez obslugiwany w aplikacji mobilnej w panelu administratora.


2. RELACJE 1 DO WIELU

Wymaganie nauczyciela:
Przynajmniej 2-3 relacje 1 do wielu.

Status:
SPELNIONE.

Zrealizowane relacje 1 do wielu:
- Category -> wiele Item
- UnitOfMeasurement -> wiele Item
- Supplier -> wiele Item
- Brand -> wiele Item
- Warehouse -> wiele Item
- Client -> wiele Order
- Worker -> wiele Order
- Order -> wiele OrderItem


3. RELACJA WIELE DO WIELU

Wymaganie nauczyciela:
Przynajmniej jedna relacja wiele do wiele, np. zamowienie - produkt przez tabele laczaca.

Status:
SPELNIONE.

Zrealizowana relacja:
Order <-> Item przez tabele laczaca OrderItem.

Opis:
Jedno zamowienie moze miec wiele produktow.
Jeden produkt moze wystapic w wielu zamowieniach.
Tabela OrderItem przechowuje pozycje zamowien.

8. URUCHOMIENIE PROJEKTU

Uruchomienie API i bazy przez Docker:
docker compose -f docker-compose-api.yml up --build

Adres API:
http://localhost:5000

Swagger:
http://localhost:5000/swagger

Uruchomienie aplikacji mobilnej:
cd SolutionOrdersMobile
npm install
adb reverse tcp:5000 tcp:5000
adb reverse tcp:8081 tcp:8081
npm start -- --reset-cache

W drugim terminalu:
cd SolutionOrdersMobile
npm run android
